style称为内部样式表,可以写整个页面的CSS
为给目标对象精准套上样式，就需要CSS选择器，通常有三种：
1.直接写标签名，如：p{
color:red
}
2.ID选择，如：#one{
color:red
}
3.class选择器，如：.hxek{
color:red
}
注※：方法2一般用于单个标签，多个标签最好不要用同一个ID，其中的“one”是用户自己定义的名称，前面记得加“#”；方法3用的最多，多个html的标签可以用同一个class，其中的“hxek”是用户自己定义的名称，前面记得加“.”。
